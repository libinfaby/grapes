Remote Desktop Software
=======================
Install **Ultraviewer** & **AnyDesk**. They are 2 among the popular remote access software providers. These softwares offers various features like unattended access, chat, file transfers etc.

`Download Ultraviewer <https://www.ultraviewer.net/en/download.html>`_

`Download AnyDesk <https://anydesk.com/en/downloads/windows>`_


DB Accessing / Editing
======================
Install **SqlDbx**. SqlDbx is an advanced SQL Editor and Database Object Explorer that can run queries, execute scripts, and browse database objects. SqlDbx is a standalone executable file, requiring no installation.

`Download SqlDbx Personal Edition <https://www.sqldbx.com/personal_edition.htm>`_
